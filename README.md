# CRUDMahasiswaADO

# Sistem Informasi Akademik Mahasiswa (SiMaster)

Aplikasi *desktop* berbasis Windows Forms untuk manajemen data mahasiswa, dilengkapi dengan fitur CRUD (Create, Read, Update, Delete), unggah foto, integrasi *database*, dan pelaporan (*reporting*).

## 💻 Teknologi yang Digunakan
* **Bahasa Pemrograman:** C# (.NET Framework)
* **Arsitektur Data:** ADO.NET (3-Tier Architecture)
* **Database:** Microsoft SQL Server
* **Laporan:** Crystal Reports
* **Installer:** Inno Setup Compiler
